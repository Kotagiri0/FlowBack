🎨 АРХИТЕКТУРНЫЕ ПАТТЕРНЫ:

1️⃣ MVC Pattern
   ├── Model: State (js/state.js)
   ├── View: HTML + CSS
   └── Controller: Managers (js/*Manager.js)

2️⃣ Module Pattern
   └── Каждый JS файл = отдельный модуль

3️⃣ Factory Pattern
   └── API.js создает запросы

4️⃣ Observer Pattern
   └── State изменения → UI обновления
